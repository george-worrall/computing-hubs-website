# Computing Hubs Educational Website

A multi-page educational website developed as part of a university team project aimed at students aged 11–19. The project focused on creating an engaging and accessible learning platform featuring courses, events, and interactive content.

## Features
- Interactive image carousel
- Searchable course and event content
- Interactive maps
- Dynamically loaded JSON data
- Responsive design using Bootstrap
- Multiple educational pages and navigation system

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- Bootstrap

## Project Overview
The website was developed collaboratively using GitHub and Visual Studio Code. The project involved responsive web design, accessibility considerations, testing, debugging, and implementing interactive front-end features.
